## 功能特点

- 实时监控论坛新帖
- 支持关键词过滤（黑名单）
- 支持关键词高亮
- 系统通知提醒
- 自定义刷新频率
- 现代化 UI 设计
- 配置持久化存储

## 使用说明

1. 主界面显示最新帖子列表
2. 点击设置按钮可以：
   - 设置爬取频率
   - 管理黑名单关键词
   - 管理高亮关键词
   - 配置通知选项

## 技术特点

- 基于 .NET 6.0
- WPF MVVM 架构
- 现代化 UI 设计
- 配置文件持久化
- 实时数据更新

## 开发环境

- Visual Studio 2022
- .NET 6.0
- Windows 10/11
